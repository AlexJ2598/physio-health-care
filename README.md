# Physio Health Care

Physio Health Care is a physiotherapy management platform built with:

* ASP.NET Core Web API
* Entity Framework Core
* SQL Server
* Angular
* Flutter

## Architecture

* API Layer
* Application Layer
* Domain Layer
* Infrastructure Layer

## Features

* Patient Management
* Appointments
* Therapy Sessions
* Payments
* Exercise Plans

## Technologies

* .NET 9
* EF Core
* SQL Server
* GitHub
* Docker (coming soon)
* GitHub Actions (coming soon)
